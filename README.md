# AI User Activity Monitoring System Using LPC2129 with HC-05

## Overview
The AI User Activity Monitoring System is an embedded systems project developed using the LPC2129 ARM7 microcontroller and HC-05 Bluetooth module. It monitors user activities through Bluetooth commands and hardware switches, evaluates user status in real time, and displays the results using an LCD and LEDs.

## Features
- Bluetooth-based wireless communication
- Real-time user activity monitoring
- UART communication support
- Push-button activity input
- LCD status display
- LED indication for user activity
- AI-like embedded monitoring logic

## Hardware Requirements
- LPC2129 Development Board
- HC-05 Bluetooth Module
- 16x2 LCD Display
- Push Button Switches
- LEDs
- Breadboard
- Jumper Wires
- Power Supply

## Software Requirements
- Keil µVision
- Flash Magic
- Bluetooth Terminal Mobile Application

## Working
1. Initialize the LPC2129, LCD, UART, and HC-05 Bluetooth module.
2. Connect a mobile phone to the HC-05 module.
3. Select a monitoring mode using the Bluetooth terminal.
4. Perform user activities through Bluetooth commands or hardware switches.
5. The LPC2129 continuously monitors and evaluates user activity.
6. The system displays activity status on the LCD, LEDs, and Bluetooth terminal.
